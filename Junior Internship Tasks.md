## 2.3 API Methods

###  transfer.create

**Purpose**: Start a transfer request, generate and send OTP via Telegram.

**Validations**:

* ext\_id must be unique

* sender\_card must be:

  * Must exists in cards table (card\_number, expiry)   
  * Active 

  * have enough balance

  * phone must exist  
* receiver\_card must exist  
* currency must be 643 or 840

**Send OTP**: Generate 6-digit OTP and send via Telegram Bot

| Request: |
| :---- |
| {     "id": 1,     "method": "transfer.create",     "params": {         "ext\_id": "tr-*{{uuid}}*",         "sender\_card\_number": "8600123412341234",         "sender\_card\_expiry": "12/26",         "receiver\_card\_number": "8600567812345678",         "sending\_amount": 15000,         "currency": 643     } } |
| **Response**: |
| {     "id": 1,     "result": {         "ext\_id": "tr-xxx",         "state": "created",         "otp\_sent": true     } } |

###  transfer.confirm

**Purpose**: Confirm transfer using OTP.

**Behavior**:

* Check ext\_id  
* Compare OTP  
* If correct: state \= confirmed, update confirmed\_at  
* If wrong: increment try\_count, block after 3 attempts

| Request: |
| :---- |
| {     "id": 1,     "method": "transfer.confirm",     "params": {         "ext\_id": "tr-xxx",         "otp": "123456"     } } |
| **Response**: |
| {     "id": 1,     "result": {         "ext\_id": "tr-xxx",         "state": "confirmed"     } } |
| {     "id": **null**,     "error": {         "message": "Incorrect OTP. Attempts left: 2"     } } |

### transfer.cancel

**Purpose**: Cancel transfer.

**Behavior**:

* If exists and in created state: mark as cancelled, update cancelled\_at

| Request: |
| :---- |
| {   "method": "transfer.cancel",   "params": {     "ext\_id": "tr-xxx"   } } |
| **Response**: |
| {   "result": {     "state": "cancelled"   } } |

###  transfer.state

**Purpose**: Return current state of a transfer.

| Request: |
| :---- |
| {   "method": "transfer.state",   "params": {     "ext\_id": "tr-xxx"   } } |
| **Response**: |
| {   "result": {     "ext\_id": "tr-xxx",     "state": "confirmed"   } } |

###  transfer.history

**Purpose**: Return list of transfers filtered by card number, date range, and status.

| Request: |
| :---- |
| {   "method": "transfer.history",   "params": {     "card\_number": "8600123412341234",     "start\_date": "2025-04-01",     "end\_date": "2025-04-21",     "status": "confirmed"   } } |
| **Response**: |
| {   "result": \[     {       "ext\_id": "tr-001",       "sending\_amount": 150000,       "state": "confirmed",       "created\_at": "2025-04-10T08:30:00"     },     ...   \] } |
| {   "result": \[\] } |
